Bereken wettelijke incassokosten (WIK). Nederlandse formule om incasso kosten te berekenen. Gebruik [incasso-berekenen] shortcode om te plaatsen.

== Description ==
De plugin om de incassokosten volgens de WIK te berekenen.  Installeer de plugin en plaatst de shortcode [incasso-berekenen] op een pagina. Hier komt dan een formulier te staan die de wettelijke incassokosten (WIK) berekend over de hoofdsom volgens de onderstaande staffel:

Hoofdsom			Kosten (% van hoofdsom)
2.500,- of minder		15 % (minimum van 40,-)
2.500,- t/m 5.000,-		10 %
5.000,- t/m 10.000,-		5 %
10.000,- t/m 200.000,-		1%
meer dan 200.000,-		0.5% (maximum van 6.775,-)

De staffel is degressief: hoe hoger de vordering, hoe lager het percentage aan incassokosten.



== Installation ==
Installeer de plugin door de zip te uplaoden in WordPress en activeer de plugin. Er zijn verder geen instellingen. Het gebruik van de shortcode op een pagina en hij werkt.
